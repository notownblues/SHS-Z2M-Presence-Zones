# SHS Z2M Presence Zone Configurator

Web-based zone configuration tool for SHS01 mmWave presence sensor integrated with Zigbee2MQTT.

## Features

- Interactive radar canvas with zone drawing
- Support for rectangular and polygonal zones
- Real-time target visualization
- Home Assistant addon integration

## Installation

Add this repository to your Home Assistant Add-on Store:

```
https://github.com/notownblues/SHS-Z2M-Presence-Zones
```
